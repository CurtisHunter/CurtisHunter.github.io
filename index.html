The Zimmer Number

  • pandas
  • requests
  • regex
  • json
  • matplotlib
  • itertools
  • data visualisation

Have you ever heard of the Bacon Number? If not, the concept is as follows: an actor who was in a movie with Kevin Bacon has a Bacon Number of 1. An actor who was in a movie with someone that has a Bacon Number of 1, has a Bacon Bumber of 2. This continues indefinitely.

Using data that I collected from the TMDB API (over 10 very long hours), I have created 'the Bacon Number' but for composers who have worked with Hans Zimmer!
